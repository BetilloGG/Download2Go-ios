Constants

kVDE_ServerFileSizeDisagreesWithExpectedFileSize

The file length advertised by the HTTP server doesn’t match the expected size. If file size validation is off, then the download proceeds anyway. If it’s on, Virtuoso aborts the download and sets the appropriate error codes.

VirtuosoConstants.h.

kVDE_InvalidMimeType

If the MIME type returned from the HTTP server doesn’t match the required MIME types whitelist you provided earlier, then Virtuoso aborts the download.
